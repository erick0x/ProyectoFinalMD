# Dashboard Pueblos Mágicos

Dashboard interactivo para análisis de indicadores y reseñas de Pueblos Mágicos de México.

## Características

- Visualización de indicadores de satisfacción por pueblo mágico
- Análisis de más de 200,000 reseñas
- Gráficas interactivas y mapa de ubicación
- Tabla de reseñas por pueblo seleccionado

## Carpeta Data

Contiene todos los notebooks y datos en formato csv

## Tecnologías

- Python
- Dash
- Plotly
- MongoDB Atlas
- Pandas

## Despliegue

Este proyecto está desplegado en Render.com
